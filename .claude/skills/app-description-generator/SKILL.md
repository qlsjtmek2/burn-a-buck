# App Description Generator (2025 ASO-Optimized)

## Purpose

Generate Google Play Store app descriptions optimized for 2025 ASO algorithms. Creates concise long descriptions (max 1,000 characters) with proper keyword density, natural language flow, and strategic structure.

## When to Use

- Creating Google Play Store long description
- ASO optimization with keyword strategy
- Preparing store listing for app launch
- Updating existing description for better rankings
- Need bilingual (Korean + English) descriptions

## Core Principles (2025 ASO Research)

### 1. Long Description Strategy
- **Character limit**: 4,000 characters (Google Play platform limit)
- **Target length**: 600-900 characters (this skill's focus)
- **Maximum length**: 1,000 characters (hard limit)
- **Keyword density**: 2.5-3% (avoid keyword stuffing penalties)
- **Mobile "Read More"**: First 250 characters are critical

### 2. Keyword Placement Priority
1. **First 250 characters**: Highest algorithmic weight + visible before "Read More"
2. **Feature descriptions**: Natural keyword distribution
3. **Category/theme sections**: Semantic keyword grouping
4. **Closing CTA**: Secondary keywords

### 3. 2025 Algorithm Updates
- **Theme-based understanding**: Context matters more than exact matches
- **Long-tail keywords**: Lower competition, higher conversion
- **Natural language**: Penalties for keyword stuffing
- **User intent focus**: Descriptive, helpful content

### 4. Writing Principles
- **No emoji**: Professional tone required
- **Clear structure**: Headers, bullets, short paragraphs
- **Benefits > Features**: Focus on user value
- **Social proof**: Stats, user testimonials (if available)
- **Scannability**: Easy to skim on mobile
